
# 🚢 Titanic Survival Prediction

## 📌 Project Overview
This project is based on the Titanic dataset and aims to predict whether a passenger survived or not using Machine Learning techniques. It demonstrates the complete Data Science workflow from data preprocessing to model evaluation.

---

## 🎯 Objective
- To analyze real-world data  
- To understand data preprocessing and visualization  
- To build a machine learning model for prediction  

---

## 🧠 Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔄 Workflow

1. **Data Collection**
   - Loaded Titanic dataset  

2. **Data Cleaning**
   - Handled missing values  
   - Removed unnecessary columns  

3. **Data Visualization**
   - Bar plots  
   - Histograms  
   - Heatmaps  

4. **Feature Engineering**
   - Created new feature: `FamilySize`  

5. **Model Building**
   - Logistic Regression  

6. **Model Evaluation**
   - Accuracy Score  
   - Confusion Matrix  

---

## 📊 Features Used
- Age  
- Sex  
- Pclass  
- Fare  
- FamilySize  

---

## 📈 Results
- Model Accuracy: **75% - 85%**  
- Confusion matrix shows good prediction performance
